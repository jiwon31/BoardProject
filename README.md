# 게시판 프로젝트

## 기능
- 회원가입, 로그인, 로그아웃
- 회원 이메일 / 닉네임 수정
- 게시글 작성, 수정, 삭제
- 파일 업로드 / 파일 다운로드
- 댓글 작성, 수정, 삭제
- 대댓글 작성, 수정, 삭제
- 게시글 조회(+페이징) + 검색 (제목, 본문 검색)
- 게시글 정렬 (최신순 / 좋아요순)
- 게시글 좋아요
- 게시글 조회수
- 내가 쓴 게시글만 조회 (+페이징)
- 좋아요한 게시글만 조회 (+페이징)
